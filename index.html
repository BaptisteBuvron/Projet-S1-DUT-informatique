Interface cerveau machine

I) Interface cerveau-machine

Une interface cerveau-machine est une interface de communication directe entre un cerveau et un dispositif externe (ordinateur, système embarqué, système électronique).

Cette communication peut permettre d’étudier l’activité du cerveau mais également d’améliorer ou de réparer des fonctions humaines.
L'enjeu des recherches dans ce domaine n'est pas de lire dans les pensées, mais de mesurer et d'interpréter l'activité neuronale en temps réel pour la traduire en commandes qu'une machine peut exécuter.
Le principe général est de permettre au cerveau de communiquer avec le monde extérieur sans passer par les chemins naturels tels que les nerfs ou les muscles.

A) Interface cerveau machine sensorielle

Une interfaces cerveau-machine sensorielles est un système dans lesquelles la machine va stimuler le cerveau. Ainsi, dans les domaines visuels, auditifs ou tactiles, des dispositifs de suppléance sensorielle sont actuellement proposés.

Exemples d'interfaces cerveau-machine sensorielles.

B) Interface cerveau machine moteur

A l’opposé, les interfaces cerveau machine motrices représentent l’ensemble des approches selon lesquelles l’activité du cerveau est transmise à un dispositif extérieur. Les signaux nerveux sont enregistrés et codés dans le but d’induire la commande de mouvements bien identifiés.
